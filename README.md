# CalcifyX 📱

**A Sleek, Scientific Calculator & Unit Converter for Android.**

CalcifyX is a high-performance tool designed for students and professionals. It combines advanced mathematical capabilities with a modern, user-friendly interface.

## ✨ Key Features

- **Smart Evaluation:** Instant results even with unclosed parentheses.
- **Scientific Mode:** sin, cos, tan (with Degree/Radian support), log, and more.
- **Unit Converter:** Seamlessly convert Length, Weight, and Temperature.
- **Adaptive UI:** Typography that shrinks for long equations to ensure readability.
- **Haptic Feedback:** Tactile vibration for a premium feel.
- **Custom Themes:** Beautiful Dark/Light modes with dynamic accent colors.

## 🛠 Built With

- **Flutter & Dart**
- **BLoC Pattern** (State Management)
- **Math Expressions Engine**
